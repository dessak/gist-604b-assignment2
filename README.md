# gist-604b-assignment2
GIST 604B Assignment 2: QGIS Desktop GIS Projects

## Overview
This repository contains the project files and output screenshots for a selection of 10 QGIS tutorials. These tutorials were completed as an assigment for the University of Arizona's GIST 604B Open Source GIS class. All tutorials are located at www.qgistutorials.com

These tutorials are intended to teach how to perform basic GIS skills using the QGIS platform. They include basic skills such as analysis, importing and modifying data, projections, as well as queries and joins.

## Software used
- QGIS Desktop 3.44.9
- Windows 11 PRO 10.0.26200 Build 26200
- No plugins utilized

## Completed Tutorials
- Making a Map
- Working with Projections
- Working with Attributes
- Importing Spreadsheets or CSV files
- Basic Vector Styling
- Basic Raster Styling and Analysis
- Performing Table Joins
- Performing Spatial Joins
- Performing Spatial Queries
- Creating Heatmaps

## Skills Demonstrated
- Utilization of QGIS Desktop
- Projecting from Geographic Coordinate Sytems
- Selecting Attributes through filters and queries
- Importing Data
- Layer styling
- Raster and vector analysis
- Spatial queries and joins
